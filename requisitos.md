# 📃 Requisitos do Sistema


## 🧑‍💼 Regras de Negócio (RN)
1. Deverá .....
2. Cada item de ....
3. Não poderá ter ....
4. ...

## ✅ Requisitos Funcionais (RF)
1. O sistema deve permitir que o usuário se cadastre com nome, e-mail e senha.
2. O sistema deve permitir o login com autenticação.
3. O sistema deve permitir que o usuário faça reservas/pedidos/etc.
4. ...

## :ballot_box_with_check: Requisitos Não Funcionais (RNF)
1. A interface deve ser responsiva.
2. O sistema deve estar disponível pelo menos 99% do tempo.
3. Os dados devem ser salvos em um banco persistente.
4. ...
